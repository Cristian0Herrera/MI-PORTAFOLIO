RefugioSOS

Aplicación web diseñada para brindar apoyo a personas refugiadas, facilitando su acceso a albergues, recursos y servicios esenciales.

El desarrollo del sistema inició con la identificación de las necesidades básicas de las personas refugiadas, seguido por el diseño de una interfaz accesible y fácil de usar. Se implementaron funcionalidades para la localización de albergues, acceso a recursos esenciales y contacto con servicios de ayuda. Finalmente, se realizaron pruebas de usabilidad y validación con usuarios para garantizar su efectividad.

Hotel Buena Vista

Sitio web informativo que presenta los servicios, instalaciones y promociones del hotel, permitiendo a los usuarios conocer sus ofertas y realizar reservaciones.

El trabajo comenzó con la recolección de información sobre los servicios, instalaciones y promociones del hotel. A partir de ello, se diseñó una estructura visual y se desarrolló un sitio web con opciones de navegación y un sistema de reservaciones en línea. Finalmente, se realizaron pruebas de compatibilidad y optimización para distintos dispositivos.
